Node学习 ​

一、安装

官网下载,安装并配置path环境变量

二、了解npm命令

​ 1..查看npm版本

​ npm -v

  1. 初始化目录

    npm init按照提示一步一步来

    或者npm init -y一步完成。

  2. 安装第三方包

​ npm install ...

  2. 卸载第三方包

​ npm uninstall ...

  1. 查看安装包列表

​ npm list

  1. 安装位置

​ npm install ... --save

​ 或者

​ npm install ... -S是安装在生产环境或线上环境

​ npm install ... --save -dev

​ 或者

​ npm install ... -D是安装在开发环境

  1. 安装cnpm淘宝的镜像

    ​ npm install -g cnpm --registry=https://registry.npm.taobao.org

    1. 查看是否正常运行

      cnpm -v

  2. 管理下载源

    npm install nrm -g

    nrm ls查看下载源列表

    ​ nrm test cnpm切换淘宝的下载源
